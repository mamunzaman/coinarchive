\# CoinArchive



Open-source platform for collecting, cataloging, submitting, and managing €2 commemorative coins.



\## Components



\### React Contributor Portal



Location:



/apps/contributor-portal



Features:



\* Contributor registration

\* Coin submission wizard

\* Draft management

\* Duplicate protection

\* EN / DE workflow

\* Dashboard and review process



\### WordPress Plugin



Location:



/wordpress/coinarchive-external-submissions



Features:



\* Submission API

\* Contributor management

\* Admin review queue

\* XLSX / CSV import

\* Polylang integration

\* ACF integration
