# CoinArchive

CoinArchive is an open-source platform for collecting, cataloging, submitting, and managing €2 commemorative coins.

## Includes
- React contributor portal
- WordPress backend plugin
- Contributor submission workflow
- Admin review queue
- Duplicate protection
- XLSX/CSV coin import
- Polylang EN/DE content workflow
- ACF-compatible coin fields
